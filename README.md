# 🌍 Climate Trend Analyzer

A comprehensive Data Science project that analyzes historical climate data to identify trends, detect anomalies, and generate insights using Python and visualization techniques.

---

## 📌 Project Overview

Climate Trend Analyzer is designed to:

* Analyze temperature, rainfall, and climate patterns over time
* Detect unusual climate anomalies
* Visualize long-term environmental trends
* Provide insights useful for research, policy-making, and sustainability planning

---

## ❗ Problem Statement

Climate change is a global challenge. Organizations need data-driven insights to:

* Monitor environmental changes
* Predict future climate patterns
* Identify abnormal climate behavior
* Support sustainable decision-making

---

## 🌎 Industry Relevance

Used in:

* Environmental Research
* Agritech & Smart Farming
* Energy & Utilities
* Climate Risk Analysis
* Government Policy Planning

Inspired by real-world systems used by organizations like NASA, NOAA, and the United Nations.

## ⚙️ Installation & Setup

### 🔹 Backend Setup

```bash
cd backend
python -m venv venv
venv\Scripts\activate   # Windows
# source venv/bin/activate  # Mac/Linux

pip install -r requirements.txt
```

### 🔹 Run Backend

```bash
python generate_sample_data.py
python main.py
```

---

### 🔹 Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
